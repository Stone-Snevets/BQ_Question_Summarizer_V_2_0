Teen Bible Quiz

Question Summary Program

Version 2.1.2
Instructions
1
Wait for the program to load up
  • There will be a message in the terminal below letting you know it's ready
2
Select a file containing official TBQ questions
  • Make sure you are inputting a PDF, TXT, DOCX, or RTF file
  • Click here for a list of rules the questions should follow
  • The program will automatically start running once it receives your file
3
Once complete, use the buttons below to:
  • View the output in your browser
  • View summary tables (note frequency, concordance, chapter analysis)
  • Download the output as a CSV file

View the Shorthand Key to understand the output fields.
