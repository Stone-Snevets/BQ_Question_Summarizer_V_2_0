Teen Bible Quiz

Question Summary Program

Instructions:

1. Wait for the program to load up
    -> There will be a message in the big black box below letting you know it's ready
    
2. Select a file containing official TBQ questions
    -> Make sure you are inputting a PDF, TXT, or DOCX file
    -> Click here for a list of rules the questions should follow
    -> The program will automatically start running once it receives your file
    
3. Once the program has completed you have the option to...
    -> View the file in your browser
    -> Download the file as a comma separated file (CSV)
    --> The downloaded file will appear in your downloads folder 
    -> The buttons to do these are below the black box
